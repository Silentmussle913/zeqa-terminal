Zeqa Terminal

A powerful tool for Minecraft and Zeqa Network enthusiasts

Getting Started

To use Zeqa Terminal, clone the repository by running the following command:

git clone https://github.com/Silentmussle913/zeqa-terminal

If you don't have Git installed, download the packaged version by clicking the button below:

Download Packaged Version

After downloading, unzip the file and navigate to the zeqa-terminal folder. Run the program by double-clicking the Start Program.bat file. It has a built-in help menu, which you can access again by running:

node adv-help

Note: You must have Node.js installed.

How to Use the Program

Below are the commands you can use with Zeqa Terminal:

  1. Get Online Players from a Minecraft Server

    Command: node mcStatusPlayers

    Description: Fetches the list of online players from the specified Minecraft server IP and port.

    Example:

    node mcStatusPlayers as.zeqa.net 10001

  2. Get Server Details

    Command: node serverdetails <server.ip> []

    Description: Fetches and displays online players, server version, MOTD, and plugins.

    Example:

    node serverdetails play.example.com 19132

  3. Get Bedrock Server Download Link

    Command: node getServerSoftware.mjs []

    Description: Opens a new tab with the download link for the specified Bedrock server version.

    Example:

    node getServerSoftware.mjs 1.18.12.01 win

  4. Check Player Status from Zeqa Network

    Command: node players

    Description: Fetches the list of online players from the specified Zeqa lobby.

    Example:

    node players as1

  5. Find Player on Zeqa Network

    Command: node find

    Description: Searches across all Zeqa regions and lobbies to check if the specified player is online.

    Example:

    node find Silentmussle913
